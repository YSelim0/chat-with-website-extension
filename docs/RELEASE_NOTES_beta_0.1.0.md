# Release Notes - 0.1.0-beta.1

## Highlights

- OpenRouter-first onboarding flow
- Dynamic model picker with free-model filtering
- Active website scanning and local page snapshots
- Grounded chat that answers from saved page context
- Conversation history with reopen support
- Archived versus live snapshot labeling

## Scope

This beta focuses on public `http` and `https` websites and uses the user's own OpenRouter API key.

## Notes

- Some free OpenRouter models may be temporarily rate-limited upstream.
- Browser-internal pages such as `chrome://` are not supported.
- API keys are stored locally in this beta without optional encryption.
