## Before Uploading

- Run `npm run check`
- Run `npm test`
- Run `npm run build`
- Run `npm run zip`
- Review `docs/RELEASE_CHECKLIST.md`

## Recommended Upload Bundle

- extension zip from `.output/`
- store listing copy from `docs/STORE_LISTING.md`
- release summary from `docs/RELEASE_SUMMARY.md`
- beta notes from `docs/RELEASE_NOTES_beta_0.1.0.md`
